The projects featured in this portfolio are taken from my time at QUT and in the workforce and range from small independent projects to large group projects.

Cane Bin Consignment Application

Myself and another 3 students were engaged in the development of a sugar cane train deployment system, aimed at automating the paper based system currently used by the majority of Australian sugar mills. This is a fullstack development project from the initial design phases in which we pitched our design to an industry rep, to the eventual full project development. This project was completed to the satisfaction of our industry contact, and it is intended this proof of concept will be pitched to Australian Sugar Mills, for potential implementation. As the university allows us to release information about this, this link will be updated with more information.

Form Generator

This form generator uses python to read an excel register, and translate desired information from the excel file into a word template. I developed this project for a company who wanted to increase the level of automation in their documentation. This program uses a simple GUI and minimises user input as requested by the client. This is an ongoing project that is developed in python using various simple python libraries.

Autofill Documents

Similar to the previous project, this used Python to generate doucments based on the information contained in an excel file. This information is transferred into a prepared word document template, and generates a new template for each value contained in the excel file. This employs multiple python libraries and uses a simple GUI that minimises user input to reduce user error as requested.

Real Time Asset Trading System

This was a large group project requiring both front and back end development knowledge. This project involved developing an asset trading system for within a hypothetical company. This project employs a GUI, database integration and was developed using test driven design principles throughout.
